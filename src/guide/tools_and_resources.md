# Tools &amp; Resources

Below are tools and resources you can use to mod Rhythm Heaven Groove.

## General
- [RHG Modding Chart](https://docs.google.com/spreadsheets/d/1fNTPlvIgDfWSvFVCRNZd32DnlALNa1PKKJYqm2MFd_8/edit?usp=sharing): A spreadsheet detailing info about each minigame

## Audio
- [CWVTool](https://github.com/conhlee/cwvtool): A command-line tool for decoding and encoding cwv audio files

## Visuals
- [TEXTool](https://github.com/conhlee/textool): A command-line tool for decoding and encoding tex texture files

## Custom Levels
- [Metronome](https://github.com/ThatZeoGal/Metronome): A command-line tool for creating wavmark (wvb) tempo files
