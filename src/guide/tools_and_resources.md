# Tools &amp; Resources

Below are tools and resources you can use to mod Rhythm Heaven Groove.

## Audio
- [cwvtool](https://github.com/conhlee/cwvtool): A command-line tool for decoding and encoding cwv audio files

## Visuals
- [pita](https://github.com/Titoot/pita): A command-line tool for decoding tex texture files

## Custom Levels
- [TempoDough](https://github.com/ThatZeoGal/TempoDough): A command-line tool for creating wavmark (wvb) tempo files
